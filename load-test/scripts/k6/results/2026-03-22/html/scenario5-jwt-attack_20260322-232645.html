JwtAuthenticationFilter(GlobalFilter order=HIGHEST_PRECEDENCE+4)는 route-level filter인 RequestRateLimiter보다 먼저 실행됩니다. 잘못된 JWT 요청은 rate-limiter의 Redis 호출 없이 GlobalFilter 단계에서 즉시 401을 반환합니다.

이는 두 가지를 의미합니다:
1. 공격 트래픽이 Redis에 부하를 주지 않음 (rate-limiter 우회가 아니라 도달 불가)
2. JWT 거절 응답(401) 속도가 Redis round-trip 없이 빠름
